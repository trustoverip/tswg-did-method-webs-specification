## IANA Considerations
This section is normative.

[[ref: KERI event streams]] use the registered [application/cesr media type with IANA](https://www.iana.org/assignments/provisional-standard-media-types/provisional-standard-media-types.xml). The media type is `application/cesr`. The registration follows the template in [RFC6838](https://tools.ietf.org/html/rfc6838).
