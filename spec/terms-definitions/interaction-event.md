[[def: interaction-event, interaction event interaction events]]

~ A key event that anchors external data to an [[ref: AID]], as defined by the [[ref: KERI specification]]. An interaction event does not change the [[ref: key state]] of the [[ref: AID]]. 

~ More detail: [WebOfTrust glossary](https://weboftrust.github.io/WOT-terms/docs/glossary/interaction-event).
