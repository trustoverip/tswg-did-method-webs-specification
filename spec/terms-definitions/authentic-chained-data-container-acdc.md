[[def: authentic-chained-data-container, authentic chained data container, ACDC, ACDCs]]

~ a variant of [[ref: the Verifiable Credential (VC) specification]] that inherits the security model derived from [[ref: KERI]], as defined by the [ACDC specification](https://trustoverip.github.io/kswg-acdc-specification/). See [[xref: keri1, authentic-chained-data-container, authentic chained data container]] in the KERISuite glossary for more detail.
