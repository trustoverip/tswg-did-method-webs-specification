[[def: BADA-RUN]]

~ Best available data acceptance - Read/Update/Nullify provides a medium level of security because events are ordered in a consistent way, using a combination of date-time and a key state. The latest event is the one with the latest date-time for the latest key state. 

More detail: [The KERI spec](https://trustoverip.github.io/tswg-keri-specification/#bada-best-available-data-acceptance-policy).
